    对多项式以及正余弦函数的一些微积分性质的初探

    ——一类函数在微积分运算上的封闭性

    大一上高数的小论文，我也不知道我写的东西有什么意义。

    在探究三角函数的微分与积分时，我发现了一件有趣的事情，即对于一个整数次的正余弦函数，它的微积分也一定可以用次数小于等于其的正余弦函数表示（积分里的常数项不考虑），或许还会跟上一些多项式，我们以\(2\)次正余弦函数为例：

    \[ \frac{\mathrm{d}}{\mathrm{d}x}(\sin^2 x)=2\sin x \cos x \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}(\sin x \cos x)=\cos^2x-\sin^2x \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}(\cos^2 x)=-2\sin x\cos x \]

    很明显在求导之后 \(\sin x\) 与 \(\cos x\) 的次数之和总是\(2\)。反之，我们对两侧同时积分，也可以发现积分的情况：

    \[ \int \sin^2 x \, \mathrm{d}x = \frac{x}{2} - \frac{1}{2}\sin x \cos x + C \]
    \[ \int \sin x \cos x \, \mathrm{d}x = \frac{1}{2}\sin^2 x + C = -\frac{1}{2}\cos^2 x + C \]
    \[ \int \cos^2 x \, \mathrm{d}x = \frac{x}{2} + \frac{1}{2}\sin x \cos x + C \]

    结果中正余弦的次数和也是二，接下来我证明这一点。

    为了方便，我记 \(\sin^ax\cos^bx=[a,b]\) ，其中 \(a, b\in \mathbb{Z}^+\)

    通过求导的乘法法则不难得到：

    \[ \frac{\rm{d}}{\mathrm{d}x}[a,b]=a\ [a-1,b+1]-b\ [a+1,b-1] \]
    \[ a+b=a-1+b+1=a+1+b-1 \]

    对于积分，我们可以利用分部积分法：

    \[ \int [a,b] \, \mathrm{d}x = -\frac{[a-1,b+1]}{a+b} + \frac{a-1}{a+b} \int [a-2,b] \, \mathrm{d}x \]

    或者也写成这样的形式：

    \[ \int [a,b]\, \mathrm{d}x = \frac{[a+1,b-1]}{a+b} + \frac{b-1}{a+b} \int [a,b-2] \, \mathrm{d}x \]

    于是可以开始递推，最终后面一半就一定是 \([1,0]\), \([1,1]\), \([0,1]\) 三种形式之一，积分也就很好求了。

    举例：

    \[ \int [4,6] \, \mathrm{d}x = -\frac{1}{10}[3,7] -\frac{3}{80}[1,7] + \frac{1}{160}[1,5] + \frac{1}{128}[1,3] + \frac{3}{256}[1,1]+ \frac{3}{256}x + C \]

    最高次是 \(10\) ，符合我之前的想法。

    又比如说：

    \[ \int [3,5] \, \mathrm{d}x = \frac{1}{8}[0,8] - \frac{1}{6}[0,6] + C \]

    （其实有奇数次项的时候不需要递推，直接换元凑积分很快就能做出来）

    于是这个结论就得证了，用数学语言来表示就是：

    \[ \forall a,b \in \mathbb{Z}^+,\,\,p\in \mathbb{R} ,\,\,\exists \, i_n,j_n \leq a+b,\,\,i_n,j_n\in\mathbb{Z^+}\,\,\,q_n,r_n,k\in\mathbb{R}\,\,\,\rm{s.t.} \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}p[a,b]=\sum q_n[i_n,j_n] \]
    \[ \int p[a,b] \, \mathrm{d}x=\sum r_n[i_n,j_n] + kx+ C \]

    考虑到积分与微分的运算是可以自由地进行加减的，所以我们可以把左边也写成求和的形式，即有：

    \[ \forall a,b \in \mathbb{Z}^+,\,\,p_n\in \mathbb{R} ,\,\,\exists \, i_n,j_n \leq a_{max}+b_{max},\,\,i_n,j_n\in\mathbb{Z^+}\,\,\,q_n,r_n,k\in\mathbb{R}\,\,\,\rm{s.t.} \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}\left(\sum p_n[a_n,b_n]\right)=\sum \frac{\mathrm{d}}{\mathrm{d}x}\left(p_n[a_n,b_n]\right)=\sum q_n[i_n,j_n] \]
    \[ \int \left(\sum p_n[a_n,b_n]\right) \,\mathrm{d}x=\sum\left(\int p_n[a_n,b_n] \, \mathrm{d}x\right)=\sum r_n[i_n,j_n] + kx+ C \]

    为了更进一步的方便，我规定：

    \[ \mathbb{SC}(n) = \left\{ \sum_{i+j \le n} c_{ij} \sin^i x \cos^j x \mid c_{ij} \in \mathbb{R} \right\} \]

    要注意的是这里的 \(\mathbb{SC}\) 并不是一个一一映射的函数，它代表了一种集合，表示一切次数小于 \(n\) 的正整数次正余弦函数之和的集合，如果是单纯的不带括号的 \(\mathbb{SC}\) ，则表示一切正余弦函数之和的集合，其中\(\rm{S,C}\)是 \(\sin\) 与 \(\cos\) 的首字母。 于是上面的结果可以改写成：

    \[ \forall f(x) \in \mathbb{SC}(n) \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}f(x) \in \mathbb{SC}(n) \]
    \[ \int f(x) + kx +C \in \mathbb{SC}(n) \]

    我感觉这是很有意思的，如果忽略积分后的一次项以及常数项，这些正余弦函数的微积分运算总是可以用有限的项表示出来，而这些项的次数也总是小于等于原函数，好像是有封闭性一样。

    不过很可惜，忽略常数项还说得过去，但是那个一次项我是怎么也忽略不了，这是因为 \([0,0]\) 作为一个常数项，其积分结果结果出现一个一次项。

    所以我想能不能有一个真正的封闭性集合呢？在某一天做练习题第65页第7题的时候我脑洞大开：是不是只需要给这些正余弦函数也乘上一个多项式就好了？多项式与正余弦函数乘积的微积分是否还是多项式与正余弦函数的乘积？

    说干就干，我开始计算。对于微分还是很好处理的：

    \[ \frac{\mathrm{d}}{\mathrm{d}x}\left( x^n[a,b] \right)= n x^{n-1}[a,b]+a x^n[a-1,b+1]-bx^n[a+1,b-1] \]

    正余弦函数的次数仍旧不变，多项式的次数也不会增加。

    对于积分，我们先来考虑一个事情，那就是任意的 \([a,b]\) 其实都可以化成 \(\sin kx\) 与 \(\cos kx\) 的和，这点用积化和差与数学归纳法就可以做到：

    我们首先定义集合 \(\mathbb{S}\)：

    \[ \mathbb{S}=\left\{\sum c_k \cos(kx) + \sum d_k \sin(kx)\mid k \in \mathbb{N},\,\ c_k,d_k \in \mathbb{R} \right\} \]

    \(\mathbb{S}\) 就是一切这些\(\sin kx\) 与 \(\cos kx\) 的线性组合。

    再看积化和差：

    \[ \cos(Ax) \cos(x) = \frac{1}{2}[\cos(A+1)x + \cos(A-1)x] \]
    \[ \sin(Ax) \cos(x) = \frac{1}{2}[\sin(A+1)x + \sin(A-1)x] \]
    \[ \cos(Ax) \sin(x) = \frac{1}{2}[\sin(A+1)x - \sin(A-1)x] \]
    \[ \sin(Ax) \sin(x) = -\frac{1}{2}[\cos(A+1)x - \cos(A-1)x] \]

    1.首先，\([1,0]\) 与 \([0,1]\) 无疑 \(\in\mathbb{S}\)

    2.当 \(\sin kx\) \(\in\rm{S}\) 时，根据上面的公式， \(\sin kx[1,0]\) 与 \(\sin kx[0,1]\) 也就一定 \(\in\mathbb{S}\)

    3.于是 \([a,b]\) 一定 \(\in\mathbb{S}\)

    反过来其实也可以证明：

    \[ c_k \cos(kx) + d_k \sin(kx) \in \mathbb{SC} \]

    因此实际上有：

    \[ \mathbb{S} \equiv \mathbb{SC} \]

    我们现在就可以把 \(x^n[a,b]\) 写成 \(x^n\sum (c_k \cos(kx) + d_k \sin(kx))\) 的形式了。

    于是积分就变得容易，只需要分别用分部积分递推：

    \[ \int x^n\cos kx \,\mathrm{d}x=\int \frac{x^n}{k} \, d\sin kx=\frac{x^n}{k}\cdot \sin kx-\int \sin kx \, \mathrm{d}\frac{x^n}{k} \]

    （\(x^n\sin kx\) 同理）

    我们就总是可以把它表示为一串（多项式乘以正余弦函数）的形式

    于是这个集合就真的在微积分上具有封闭性了。

    我们定义：

    \[ D(x)=\sum_{i=0}^n k_i x^i \]
    \[ SC(n)=\sum_{i+j \leq 0} k_{ij}[i,j] \]
    \[ SCD(n,x)=\sum SC(n)D(x) \]

    其中 \(D(x)\) 表示多项式之和 \(SC(a,b)\) 表示正余弦函数之和， \(SCD(n,x)\) 就表示一切为正余弦函数×多项式之和形式的函数。

    定义：

    \[ \mathbb{SCD}(n,x) = \left\{SCD(n,x)\mid n,x\in \mathbb{N}\right\} \]

    换言之，\(\mathbb{SCD}\) 是所有形如 \(D(x)\sin^ax\cos^bx\) 的函数的有限线性组合，在不含括号的情况下 \(\mathbb{SCD}\) 没有次数的限制。

    于是有定理：

    \[ \forall f(x) \in \mathbb{SCD}(n,x) \]
    \[ \frac{\mathrm{d}}{\mathrm{d}x}f(x) \in \mathbb{SCD}(n,x) \]
    \[ \int f(x) \,\mathrm{d}x \in \mathbb{SCD}(n,x) \]

    示例，求 \(x^3\sin^2 x\cos^5 x\) 的积分：

    \[ \begin{aligned} \int x^3 \sin^2 x \cos^5 x \, \mathrm{d}x = \frac{1}{64} \Bigg[ & \\ & 5\left( (x^3-6x)\sin x + (3x^2-6)\cos x \right) \\ & - \left( (\frac{x^3}{3} - \frac{2x}{9})\sin 3x + (\frac{x^2}{3} - \frac{2}{27})\cos 3x \right) \\ & - 3\left( (\frac{x^3}{5} - \frac{6x}{125})\sin 5x + (\frac{3x^2}{25} - \frac{6}{625})\cos 5x \right) \\ & - \left( (\frac{x^3}{7} - \frac{6x}{343})\sin 7x + (\frac{3x^2}{49} - \frac{6}{2401})\cos 7x \right) \\ \Bigg] + C \end{aligned} \]

    没有问题，最后的结果确实是一堆正余弦函数乘以多项式之和的形式。

    此外，不难证明对于这个 \(\mathbb{SCD}\) 集合，加法、减法、乘以一个数字这些运算也是具有封闭性的.

    一些更大胆、不切实际、莫名其妙的想法

    1.在上述的讨论中，有一个前提是没有变的，那就是次数一定是正整数次，如果说把这个结论推广到正有理数、正实数、乃至实数的范围里，还会有类似的结论吗？

    2.上面这些式子总是可以表示成多项式乘以正余弦函数的形式，而且除非是零次，这两个函数是不能够通过微积分运算互相得到的，那么能不能说我把多项式和正余弦函数理解成坐标轴？我想，不计系数，如果我把 \(x\) 轴替换为 \(\sin x\) 的次数， \(y\) 轴替换为 \(\cos x\) 的次数， 把 \(z\) 轴替换为 \(x\) 的次数，那么每一个函数就都可以对应唯一一个空间内的点集，只不过目前来说这些点都是整点，那么一个函数在微分或积分后，它对应的点集会有什么改变吗？

    3.结合 1 与 2 ，如果可以对应非整点了，那么点集又会是什么样的？

    这一部分的内容已经大大超出我的能力范围了，我想我大概是在说胡话，但还是尝试算下吧。

    建立一个平面直角坐标系，\(x^a \sin^b x\cos^c x\) 映射为其上的一个点 \((a,b,c)\) ，其上任意的一点 \((a,b,c)\) 就表示 \(x^a \sin^b x\cos^c x\) 。

    首先考虑微分，计算得：

    \[ \frac{\mathrm{d}}{\mathrm{d}x} x^a \sin^b x\cos^c x = ax^{a-1}\sin^b x\cos^c x+bx^a\sin^{b-1}x\cos^{c+1}x-cx^a\sin^{b+1}x\cos^{c-1}x \]

    对应到点：

    \[ \frac{\mathrm{d}}{\mathrm{d}x}(a,b,c)=a(a-1,b,c)+b(a,b-1,c+1)-c(a,b+1,c-1) \]

    于是我们有结论:

    \[ 微分运算使此空间上的点向\, -x \,以及\, y=z \,三个方向分裂一个单位 \]

    考虑边界条件，即 \(a,b,c\geq 0\)，我们可以料想到在多次微分后，点 \((a,b,c)\) 将多次分裂，最终的点集是：

    \[ y+z=b+c \,\,\, (0 \leq x\leq a)\,\,\,(0 \leq y \leq b+c)\,\,\,(0 \leq z \leq b+c) \,平面上的整点 \]

    形象描述的话就是，\((a,b,c)\) 在分裂的过程中逐渐流向 \(yOz\) 平面，并向 \(y=-z\) 方向的两侧分散。

    我画了几张图来具象化表示（红点是原函数，蓝点是一阶微分，绿点是二阶微分）：

    接着考虑积分，因为积分十分困难，我不觉得这东西会有一个表达式，所以直接套用上述的论证（即 \(\mathbb{SCD}\) 集合的性质），只不过在这个过程中，高次积分的结果不仅仅是平面，通过直觉我认为是：

    \[ y+z \leq b+c \,\,\, (0 \leq x\leq a)\,\,\,(0 \leq y \leq b+c)\,\,\,(0 \leq z \leq b+c) \,空间上的整点 \]

    形象描述的话就是, \((a,b,c)\) 在积分的过程中也在分裂，不过这次会逐渐流向原点，\(y+z=b+c\) 与 \(x=a\) 是两个边界。
